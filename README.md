# functions

cubebroot
```py
import mathena
print(mathena.cuberoot(27))
```

prime factorization

```py
import mathena
print(mathena.prime_factorization(27))
```

is a perfect square

```py
import mathena
print(mathena.is_perfect_square(4))

```

is a perfect cube

```py
import mathena
print(mathena.is_perfect_cube(27))

```
